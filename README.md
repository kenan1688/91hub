# 91Hub Demo

一个静态前端发布页模板，包含地址发布、搜索、分类筛选、推荐列表、公告弹窗和明暗主题切换。

## 本地预览

直接用浏览器打开 `index.html` 即可。

## GitHub Pages

推送到 GitHub 后，可以在仓库的 `Settings -> Pages` 中选择：

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

保存后等待 GitHub Pages 部署完成。

## 文件

- `index.html`：页面结构
- `styles.css`：样式和响应式布局
- `app.js`：列表数据、筛选、搜索、弹窗和复制交互
